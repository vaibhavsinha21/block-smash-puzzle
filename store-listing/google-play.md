🧩 BLOCK SMASH PUZZLE — The Ultimate Block Puzzle Game!

Drag and drop colorful blocks onto the 8×8 board, complete rows and columns to clear them, and rack up massive scores with epic combos!

• Drag blocks from the tray onto the board
• Fill a complete row or column to clear it
• Plan ahead — once blocks are placed, they stay!
• Use the undo button if you make a mistake
• Place all 3 blocks to get a new set
• Game ends when no pieces can fit — survive as long as you can!
